# DJ Playlists (Local)

This directory is reserved for local, environment-specific playlist exports.

Historical, absolute-path playlists have been moved to:
- `archive/dj_playlists_local/`

Generate new playlists into `artifacts/` (or another local path) and avoid
committing absolute library paths into `config/`.
