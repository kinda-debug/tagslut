# Setup & Configuration

## Supported Language Versions

This SDK is compatible with the following versions: `Java >= 1.8`

## Installation

If you use Maven, place the following within the _dependency_ tag in your `pom.xml` file:

```XML
<dependency>
    <groupId>com</groupId>
    <artifactId>tagslutapisdk</artifactId>
    <version>1.0.0</version>
</dependency>
```

If you use Gradle, paste the next line inside the _dependencies_ block of your `build.gradle` file:

```Gradle
implementation group: com, name: tagslutapisdk, version: 1.0.0
```

If you use JAR files, package the SDK by running the following command:

```shell
mvn compile assembly:single
```

Then, add the JAR file to your project's classpath.

## Setting a Custom Timeout

You can set a custom timeout for the SDK's HTTP requests as follows:

```java
import com.tagslutapisdk.TagslutApiSdk;
import com.tagslutapisdk.config.TagslutApiSdkConfig;

public class Main {

  public static void main(String[] args) {
    TagslutApiSdkConfig config = TagslutApiSdkConfig.builder().timeout(10000).build();
    TagslutApiSdk tagslutApiSdk = new TagslutApiSdk(config);
  }
}

```

# Sample Usage

Below is a comprehensive example demonstrating how to authenticate and call a simple endpoint:

```java
import com.tagslutapisdk.TagslutApiSdk;
import com.tagslutapisdk.exceptions.ApiError;

public class Main {

  public static void main(String[] args) {
    TagslutApiSdk tagslutApiSdk = new TagslutApiSdk();

    try {
      Object response = tagslutApiSdk.auth.introspectToken();

      System.out.println(response);
    } catch (ApiError e) {
      e.printStackTrace();
    }

    System.exit(0);
  }
}

```
