# Beatport Genres and Sub-Genres

1 month ago Updated

FollowNot yet followed by anyone

Listed below are the electronic genres currently accepted via delivery to Beatport, including the proper XML notation for each. For the full list of genres and subgenres which include the new open format genres (without XML notation), refer to [this page](https://greenroomsupport.beatport.com/hc/en-us/articles/41043520429076-Beatport-Genres-Including-NEW-Open-Format-Genres).
 

You may deliver valid sub-genres to your tracks now via XML. Or, if you wish to assign a sub-genre to a particular track/release, or if you need to update a genre after delivery, please reach out to your Label Manager.

Our system will not update genres via XML update.

**Please note that **Bass / Club** is only available for use by our Curation team and **cannot** be delivered as a primary genre via XML.*  


**also note that all XML notation listed here is for **non-DDEX** xml. If you need assistance with your DDEX genres, please reach out to submissions@beatport.com*
