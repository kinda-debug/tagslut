# Legacy Assets

This branch keeps the active V2 code and docs clean. Legacy scripts, one-off tools, and operator notes were moved under `legacy/` or removed if they were generated artifacts (logs, reports, epoch DBs).

- `legacy/tools/`: Archived v1/v2 helper scripts (kept for reference)
- `legacy/scripts/`: One-off utilities and ad-hoc scripts
- `legacy/docs/`: Old notes and drafts
- `legacy/configs/`: Local config/examples that should not be in the main root

If you need something from legacy, pull it back explicitly.
